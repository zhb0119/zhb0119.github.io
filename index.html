Horizon RDK X3 Robot System | Infineon TC264 High-Speed Vision
[智慧医疗] 基于 地平线 RDK X3 与 ROS2 构建的人机协同平台。部署轻量化 YOLO/MobileNet 实现二维码与障碍物识别，结合数字孪生交互，在 5×5m 场景中完成自主导航、巡检与物资配送。
[镜头组] 基于 英飞凌 TC264/297 多核 MCU 开发的 3–5m/s 高速视觉循迹系统。
  • 视觉感知：设计动态阈值与中值滤波流水线，实时处理 120×188 灰度图像，在复杂光照下精准提取赛道边线。
  • 控制决策：采用多核任务分配架构（图像/控制分离），基于偏差计算与 PID 舵机闭环控制，实现高速稳定过弯。
关键词：Horizon RDK X3 · ROS2 · Infineon TC264 · High-Speed Vision · PID Control · Multi-core
[Smart Healthcare] A human-robot collaboration platform based on Horizon RDK X3 and ROS2. Deploys lightweight YOLO/MobileNet for object detection, integrating digital twins for autonomous navigation and inspection in a 5×5m scenario.
[Camera Group] High-speed (3–5m/s) vision tracking system based on Infineon TC264/297 multi-core MCU.
  • Vision Perception: Designed an image processing pipeline (dynamic thresholding, median filtering) for real-time 120×188 frame analysis and robust lane extraction.
  • Control & Integration: Implemented multi-core task scheduling (vision/control separation) and PID closed-loop servo control for stable high-speed maneuvering and obstacle handling.
Keywords: Horizon RDK X3 · ROS2 · Infineon TC264 · High-Speed Vision · PID Control · Multi-core

简易自瞄装置
Simple Self-Aiming Device
设计并制作一套具备自动寻迹与激光自瞄能力的小车系统：沿黑色轨迹自动行驶（可设 1–5 圈、单圈 ≤20s），在指定或任意位置自动搜索并识别靶标，通过二维云台控制激光精准命中靶心（误差 ≤2cm）。
  • 主控：TI MSPM0 系列 MCU（如 MSPM0G3507）；视觉：K230（用于图像识别）。
  • 寻迹：八路灰度/红外阵列，支持直道跟踪与 90° 转弯控制，按键设置圈数，完成后自动停止。
  • 自瞄：双轴云台（步进/舵机），激光与摄像头同轴；OpenCV 矩形检测 + 透视变换定位靶心，PID 控制闭环对准。
  • 工程细节：数模隔离供电、云台高刚性结构（常用 3D 打印）、双独立电源开关规则满足。
关键词：MSPM0G3507 · K230 · 8-channel grayscale · 2-DoF gimbal · OpenCV · PID · Coaxial laser
Built an autonomous line-following car with a 2-DoF gimbal laser aiming module. The system completes 1–5 laps (≤20s per lap), detects/locks targets automatically, and aligns the laser to the bullseye with ≤2cm error.
  • MCU: TI MSPM0 (e.g., MSPM0G3507); Vision: K230 for target perception.
  • Line following: 8-channel grayscale/IR array with straight tracking and 90° turn handling; lap count configurable; auto stop.
  • Aiming: dual-axis gimbal (stepper/servo), coaxial camera+laser; OpenCV detection + homography; PID closed-loop alignment.
  • Engineering: mixed-signal power isolation, rigid gimbal (3D-printed), dual power-switch rule compliance.
